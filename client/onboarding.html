BellForge Pairing Console

Create a pairing session, display a short-lived code and QR payload, then claim and complete registration from a signed-in user token.

Routes: /api/devices/pairing/init, /api/devices/pairing/claim-code, /api/devices/pairing/claim-qr, /api/devices/pairing/status.

Device Pairing Init

Use this on unauthenticated devices. The server returns a single-use code plus a signed token for QR flow.
